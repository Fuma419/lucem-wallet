# Vercel Deployment

**Trigger:** Agent modifies build config, `vercel.json`, or needs to deploy.

## Config
- `vercel.json`: `buildCommand: "npm run build"`, `outputDirectory: "build/"`, SPA rewrites.
- `package.json` `engines.node: "20.x"` — prevents Vercel auto-upgrading.
- Secrets auto-generated by `utils/build.js` from `secrets.testing.js`.
- Token: `$VERCEL_TOKEN`, scope: `my-team-5c660a1c`, project: `lucem-wallet`.

## Commands
```bash
vercel deploy --prod --token $VERCEL_TOKEN --scope my-team-5c660a1c --yes
```

## Rules
1. New HTML entry points → add rewrite in `vercel.json`.
2. New React Router routes → add rewrite.
3. Never add `build/` to git.
